# docs/README.md Optional Template

## Copy This As

`docs/README.md`

## When To Use

Use this only when your repository has started accumulating stable workflows, methods, operating rules, or repository-level guidance that should live outside a single task or feature area.

## Template

```md
# docs Guide

`docs/` stores repository-level methods, rules, workflows, and stable guidance.

Use `docs/` when content:

- can be reused across multiple tasks
- serves as a stable reference
- explains a repository-level workflow or operating rule

Do not put these here first:

- task-specific notes
- early drafts
- execution artifacts

## Current Structure

## Reading Order

1. Read `docs/README.md`.
2. Enter the relevant subdirectory guide.
3. Read the specific workflow or rule document you need.
```
