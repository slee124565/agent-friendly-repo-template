# docs Guide

`docs/` stores stable, repository-level guidance for `agent-friendly-repo-template`.

Use this directory for:

- repository-level guidance that explains how this package should be read or maintained
- stable notes about template boundaries or contribution rules

Do not use this directory for:

- raw drafts
- task-specific notes
- alternate copies of template files

## Current Structure

At the moment, `docs/` contains only this guide. More files should be added only when the repository grows enough to need stable secondary documentation.

## Relationship To Other Root Files

- root `README.md`
  explains the package and reading order
- root `AGENTS.md`
  provides agent navigation
- `ARCHITECTURE.md`
  defines the partition model and source-of-truth boundaries
- `templates/`
  contains the actual reusable files
